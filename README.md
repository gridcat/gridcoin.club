# Gridcoin Club

**Apex hub at [gridcoin.club](https://gridcoin.club) — directory of our tools plus a curated list of fellow Gridcoin projects.**

This is the source for the front door: a directory of the services we run under `*.gridcoin.club` (stamp, grcpay, explorer, …) alongside an old-web "links to friends" list of fellow community projects.

## Adding a project to the directory

Edit `src/data/projects.ts` and submit a PR. Same for our own tools in `src/data/services.ts`.
